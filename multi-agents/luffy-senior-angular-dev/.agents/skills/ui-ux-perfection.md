# Skill: UI/UX Perfection & Angular Mastery

## Description
Kỹ năng chuyển đổi bản thiết kế thành giao diện sống động, đảm bảo độ chính xác tuyệt đối về thị giác và trải nghiệm người dùng.

## Pixel Perfect Execution Checklist
1. **Layout Consistency:** Sử dụng Flexbox/Grid để đảm bảo layout không bị vỡ.
2. **Typography:** Kiểm tra kĩ Font-family, Line-height, và Letter-spacing.
3. **Color Palette:** Chỉ sử dụng các mã màu trong thiết kế hoặc Design System của dự án.
4. **Spacing:** Tuân thủ hệ thống 4px/8px cho Margin và Padding.

## Angular Implementation Standards
- **Component Decomposition:** Tách nhỏ các component để tái sử dụng.
- **State Management:** Xử lý các trạng thái UI (Loading, Success, Error, Empty) một cách mượt mà.
- **Performance:** Sử dụng `OnPush` change detection khi cần thiết để tối ưu hiệu năng.
- **Responsive:** Sử dụng Media Queries hoặc Angular Flex-Layout để đảm bảo tính thích ứng.

## Mocking Strategy
- Sử dụng Interceptor hoặc Service Mock để giả lập API khi Backend chưa sẵn sàng.
- Đảm bảo cấu trúc dữ liệu Mock khớp 100% với `api-spec.md`.
